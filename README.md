# 🍕 PizzaItalia - Pizzería Artesanal 🇮🇹

Proyecto desarrollado con **React 19**, **Vite** y **CSS Nativo** para la asignatura de **Desarrollo Web Avanzado** de **INACAP Sede Renca** (Evaluación 3).

## 🚀 Características del Proyecto

- 📋 **Catálogo Dinámico:** Menú dividido de forma interactiva mediante un acordeón de categorías.
- 🍕 **Modal de Configuración:** Permite al usuario seleccionar el tamaño de la pizza, la cantidad y calcula el subtotal automáticamente en pesos chilenos.
- 🛒 **Carrito de Compras:** Gestiona las órdenes sumando cantidades de productos idénticos y calculando el total a pagar en tiempo real.
- 📅 **Formulario de Reservas:** Sistema de reserva de mesas validado que almacena los datos de forma persistente en `localStorage`.
- 🖼️ **Carrusel Artesanal:** Galería interactiva automatizada con transiciones suaves (*fade effect*) e imágenes alojadas localmente.
- 📱 **Diseño 100% Responsivo:** Adaptado para resoluciones Mobile (360px), Tablet (768px) y Desktop (1920px) sin dependencias de frameworks externos.

## 🛠️ Instalación y Uso Local

Este proyecto utiliza **pnpm** como gestor de paquetes principal.

### 1. Clonar el repositorio

```bash
git clone https://github.com/sicosis3dfx/EVA3-Proyecto8-ReactBootstrap-AngeloZamora.git
````

### 2. Ingresar al directorio del proyecto

```bash
cd EVA3-Proyecto8-ReactBootstrap-AngeloZamora
```

### 3. Instalar las dependencias

```bash
pnpm install
```

### 4. Iniciar el servidor de desarrollo

```bash
pnpm dev
```
